Automated E-Currency exchange

Use this form to perform automated, real time exchanges between currencies. Most exchanges are processed instantly after payment, unless otherwise noticed when you select the pair to be exchanged.

From currency: Select the e-currency type you would like to exchange from.

To currency: Select the e-currency type you would like to receive in exchange.

Amount to exchange: Enter the amount of currency you would like to send for this exchange.

Destination account: Enter the account where you would like to receive your funds resulting from this exchange.

Your email: Enter your email address. Make sure you enter your email address correctly. Important information regarding this order will be sent to this email address.

Exchange rate: This box is updated real time with the exchange rate for the selected combination. If you receive an "N/A" message on this box, the selected exchange combination is not available at this time.

You receive: This is the amount of the "To currency" you will receive as a result of this exchange
